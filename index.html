Z3-Falcon

Z3-Falcon is a derived SMT solver based on Z3. It supports the following logics: QF_BV, QF_UFBV, QF_ABV, QF_AUFBV, QF_FP, QF_BVFP It is a project mainly developed in the Automated Reasoning Group, Zhejiang University, Hangzhou, China.
